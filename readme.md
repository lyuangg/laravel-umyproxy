# laravel-umyproxy

umyproxy laravel package.

### Install

```
composer require 'lyuangg/laravel-umyproxy'
```


### Usage

1. Modify laravel configuration.

.env

```
DB_SOCKET=/tmp/umyproxy.socket
```

2. start umyproxy

```
php artisan umyproxy:serve
```
